## [0.5.0] - 2025-12-07

### Added
- AI 收支建議功能（使用 Cloudflare Workers AI）。
- 版本更新時顯示更新日誌彈窗。
- 所有檔案加入正體中文功能註解。
- README.md 完整重寫，包含專案結構與依賴關係說明。

### Changed
- 優化 Vite 建構配置，使用 code splitting 減少 chunk 大小。
- AI 建議支援 Markdown 渲染。

## [0.4.0] - 2025-12-07

### Added
- 帳戶分組功能，可折疊展開
- 支出標籤排行圓餅圖
- 收支報表支出分類排行新增圖表切換（列表/圓餅圖）

### Changed
- Toast 提示訊息置頂並放寬寬度
- 信用卡/虛擬帳戶不計入群組總額
- 分類圓餅圖百分比與 tooltip 修正

### Fixed
- 修正帳戶分組金額計算問題

## [0.3.0] - 2025-12-06

### Added
- 掃描電子發票功能（支援台灣電子發票雙 QR Code）

- 帳戶區分實體/虛擬帳戶

- 調整帳戶卡片佈局
- UI 適應網頁版
- 新增交易頁面的分類、商家、標籤改用 InputSelect 組件
- 調整主畫面背景

### Fixed
- 修正編譯失敗問題
- 修正手機拖曳行為
- 修正手機版帳戶卡片無法拖曳問題
